## Bubble Sort (Brief Description)
Bubble Sort is a simple comparison-based sorting algorithm.  
It works by repeatedly comparing adjacent elements of the array and swapping them
if they are in the wrong order. After each pass, the largest element moves to its
correct position at the end of the array.
